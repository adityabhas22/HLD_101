# HLD 101

This repository contains my notes, code, and small experiments as I learn and explore High Level Design (HLD) concepts.

The goal is to build a growing collection of simple examples around core system design topics such as:

- load balancing
- hashing and consistent hashing
- scaling patterns
- distributed systems concepts
- other HLD topics as I continue learning

I will keep adding to this repository over time as I explore more ideas, patterns, and implementations.
